What it's JavaScript?

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. Where HTML and CSS are languages that give structure and style to web pages, JavaScript gives web pages interactive elements that engage a user.

JavaScript Datatypes

In programming, data types is an important concept.

To be able to operate on variables, it is important to know something about the type.

Without data types, a computer cannot safely solve this:

JavaScript variables can hold different data types: numbers, strings, objects and more:

let length = 16; // Number

let lastName = "Johnson"; // String

let x = {firstName:"John", lastName:"Doe"}; // Object

Handling with data types

For example, let's see the following code

let x = 16 + "JS";

Does it make any sense to add "JS" to sixteen? Will it produce an error or will it produce a result?

JavaScript will treat the example above as:

let x = "16" + "JS";

Conclussion: If you add a number and a string JavaScript will treat the number as a string.

Let's see another interesting example of JavaScript code.

let x = 16 + 4 + "JS";

Result

20JS

Conclussion: if you add two or more numbers at first and then add a string, JavaScript will sum the numbers and will add it to the string

But... If we try the previous example but adding the numbers at last?

let x = "JS"+ 16 + 4 ;

In this case, the result will be the following

JS164

Conclussion: if the first operand is a string, all operands are treated as strings.

JavaScript types are Dynamic

JavaScript has dynamic types. This means that the same variable can be used to hold different data types:

let x; // Now x is undefined

x =105; // Now x is a Number

x = "Slade Wilson"; // Now x is a String

String

You can use quotes inside a string, as long as they don't match the quotes surrounding the string:

let a1 = " It's alright "; // Single quote inside double quotes

let a2 = " He is called 'Slade' "; // Single quotes inside double quotes

let a3 = ' He is called "Slade" '; // Double quotes inside single quotes

Numbers

Extra large or extra small numbers can be written with scientific (exponential) notation:

let a = 568e5; //56800000

let b = 568e-5 //0.00568

Boolean

Booleans can only have two values: true or false

let a = 8;

let c = 1;

( a != c ) //Returns true

( a == c ) //Returns false

Booleans are often used in conditional testing.

Arrays

JavaScript arrays are written with square brackets.

Array items are separated by commas.

const villains = [ " Deathstroke ", " Scarecrow " , " Black Manta " ];

Array indexes are zero-based, which means the first item is [0], second is [1], and so on.

Objects

JavaScript objects are written with curly braces {}.

Object properties are written as name:value pairs, separated by commas.

const person = {firstName:"Slade", lastName:"Wilson", age:40, eyeColor:"brown"};

JavaScript Syntax

JavaScript syntax is the set of rules, how JavaScript programs are constructed.

Create Variables

Variables are containers for storing data (storing data values).

There are 4 Ways to Declare a JavaScript Variable:

  • Using var
  • Using let
  • Using const
  • Using nothing

Using var

var firstNumber = 10;

var secondNumber = 8;

var result = firstNumber * secondNumber;

Using let

let firstNumber = 10;

let secondNumber = 8;

let result = firstNumber * secondNumber;

Using const

In this case, const is a constant so his value will never change

const PI = 3.14159;

Using nothing

firstNumber = 10;

secondNumber = 8;

result = firstNumber * secondNumber;

JavaScript Identifiers

Identifiers are JavaScript names.

Identifiers are used to name variables and keywords, and functions.

The rules for legal names are the same in most programming languages.

A JavaScript name must begin with:

  • A letter (A-Z or a-z)
  • A dollar sign ($)
  • Or an underscore (_)

Subsequent characters may be letters, digits, underscores, or dollar signs.

Numbers are not allowed as the first character in names.

This way JavaScript can easily distinguish identifiers from numbers.
